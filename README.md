# Haptic Lab —— 触觉实验室

~~**让你的每一根神经，都臣服于律动的颤抖。**~~

Haptic Lab 是一款基于低音频率同步的安卓底层触感增强应用。它不只是一个播放器，它是一场将数字音频转化为物理欲望的炼金术。通过精准的频率提取，它能将每一声鼓点、每一段低吟，转化为直击灵魂的细腻冲击。

---

## 核心能力

- **核心震动引擎**：突破安卓标准限制，通过 `VibrationEffect` 结合动态振幅算法，模拟 255 级阶梯式感官反馈。
- **音频感官同步**：内置实时 FFT（快速傅里叶变换）分析器，精准捕捉音频中的能量峰值，实现音振合一。
- **极致平滑视觉**：基于 64 位采样点的实时贝塞尔曲线波形图，视觉律动与身体快感同步高潮。
- **多维度激发源**：内置“玫瑰海岸”、“激情澎湃”等经典预设频率，亦支持导入私人珍藏的本地音频。

## 深度使用指南

本应用针对高灵敏度反馈进行了特殊调优，建议将设备放置于敏感部位享用

1. **获取能量**：点击 `+` 按钮。选择精心挑选的预设频率，或导入你最爱的本地音源。
2. **状态确认**：当屏幕闪烁着“准备好迎接高潮了吗？”时，调整呼吸，做好准备。
3. **沉溺其中**：按下启动键。随着动态波形的起伏，感受低频脉冲对感官的侵蚀。
4. **调教控制**：进入设置中心，解锁“极致模式”倍率限制。小心，这可能会让你的马达（和身体）由于过热而尖叫。

## 技术实现 (Technical Specs)

- **频率提取**：实时分析 `1~12Hz` 低音频段能量，通过智能动态频率补偿算法，确保震感扎实不发虚。

## 风险声明

- **硬件负载**：极致模式会大幅度提升马达占空比。长时间高强度使用可能导致硬件发热或寿命损耗。
- **生理提醒**：过度沉溺于模拟律动可能导致感官疲劳。请根据个人承受能力适度调节倍率。
