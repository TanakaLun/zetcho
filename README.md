# Haptic Lab 🧬

**让你的每一根神经都随律动颤抖。**

Haptic Lab 是一款基于低音频率同步的安卓触感增强应用。它不仅仅是一个另类的播放器，它能将你最爱的音频文件转化为细腻、强力的物理反馈，让你在音律中抵达顶点……啊，我要去了！

## 🌟 核心功能

- **音频震动同步**：利用 FFT（快速傅里叶变换）算法实时提取低音能量，将其转化为手机马达的物理冲击。
- **沉浸式视觉**：动态贝塞尔曲线波形，视觉与触觉同步高潮。
- **自定义导入**：支持任何本地音频，无论是重低音电音还是娇喘低吟，都能带给你最直观的反馈。

## 🍑 深度体验

本软件针对高频响应进行了优化，建议将手机放置于敏感区域。

- **准备阶段**：点击 `+` 按钮导入你精心挑选的音源。当状态显示“准备好迎接高潮了吗？”时，一切就绪。
- **绝顶时刻**：按下播放键，感受低频脉冲对身体的侵蚀。随着波形起伏，你会发现自己已无法自拔，“もうすぐ絶頂を迎えます……”

## 🛠 技术细节

- **触感引擎**：采用 `VibrationEffect.createOneShot` 结合动态振幅控制，模拟 255 级细腻力度。
- **低音提取**：实时分析 `1~8Hz` 频段能量，确保震动精准打击在鼓点之上。

## ⚠️ 声明
本软件仅供个人娱乐使用。长时间高强度使用可能导致手机马达发热或身体过度疲劳，请适度调情。
